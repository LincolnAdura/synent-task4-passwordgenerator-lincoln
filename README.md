## Description
A command-line Password Generator built in Python that generates 
strong and secure random passwords based on user preferences and 
configuration.

## Features
- 🔐 Generates strong random passwords
- 📏 Custom password length (8-50 characters)
- 🔠 Optional uppercase letters
- 🔡 Optional lowercase letters
- 🔢 Optional numbers
- ⚡ Optional special characters
- ✅ Validates all inputs
- 🔄 Generate multiple passwords in one session

## How to Run
Make sure Python is installed, then run:
```bash
python password_generator.py
```
